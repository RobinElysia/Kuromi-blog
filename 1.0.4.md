# 1.0.4

发布日期：2026-03-24

## 本次目标
- 保持原有配色和背景氛围。
- 主页时间轴改为直线简约风。
- 重写帖子编辑、发布、详情展示链路。
- 使用成熟轮子实现 Markdown / LaTeX / Mermaid / 图片插入。
- 保持旧接口与旧数据可兼容。

## 依赖变更
- 新增 `bytemd`、`@bytemd/react`
- 新增 `@bytemd/plugin-gfm`、`@bytemd/plugin-math`、`@bytemd/plugin-mermaid`、`@bytemd/plugin-highlight`
- 新增 `class-variance-authority`、`clsx`、`tailwind-merge`、`lucide-react`

## 文件改动

### `src/components/Home.tsx`
- 主页右侧滚动器由弧线交互改为直线交互（直线轨道 + 拖拽节点 + 关键刻度按钮）。
- 帖子主区改为直线时间轴布局（纵向主线 + 节点 + 卡片），风格更简约。
- 保留原有业务逻辑：拉取帖子、搜索过滤、管理员入口、访客说明、切换壁纸与退出。

### `src/components/PostEditorPage.tsx`
- 重写为 ByteMD 编辑器方案，编辑与预览统一，支持分栏模式。
- 保留原有发帖/更新/删除 API 流程与权限判断（仅 RobinElysia、Meow 可管理）。
- 保留并增强：`.md` 导入、标签编辑、图片插入（文件按钮 + 编辑器上传接口）。
- 保持历史帖子兼容：仍使用 `content/title/tags` 字段，无需迁移旧数据。

### `src/components/PostDetail.tsx`
- 详情页结构重写为更规整的卡片式布局，保持原配色体系。
- 正文渲染统一走 `MarkdownRenderer` 新内核，避免编辑页与详情页显示不一致。
- 保留评论展示逻辑与回退逻辑。

### `src/components/MarkdownRenderer.tsx`
- 从手写 `react-markdown` 管线切换为 `ByteMD Viewer`。
- 与编辑器共享同一插件配置：GFM、数学公式、Mermaid、代码高亮。
- 新增 sanitize 策略入口（基于 ByteMD sanitize 机制扩展）提升渲染安全性。

### `src/lib/markdown.ts`
- 新增 Markdown 渲染中心配置：统一插件列表与 sanitize 策略。
- 统一 KaTeX、Mermaid 安全相关参数，避免各页面各自维护。
