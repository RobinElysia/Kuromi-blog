# 1.0.1

## 本次目标
- 保持既有配色与背景系统。
- 增加顶部导航栏（左：logo + 站点名；右：模糊搜索框、主页、帖子、标签、友链）。
- 将帖子发布能力从主页抽离到独立编辑页面。
- 编辑页面支持：发布、编辑、删除、文中插图、本地导入 Markdown。
- 帖子发布后支持 Markdown / LaTeX / Mermaid 渲染。

## 修改文件与内容

### src/App.tsx
- 新增路由分发能力（轻量 history 路由）：
  - `/` 与 `/posts` -> 主页
  - `/posts/:id` -> 帖子详情
  - `/editor` -> 帖子编辑页

- 新增全局搜索状态并透传给导航、主页、编辑页。
- 登录后统一接入顶部导航栏。

### src/components/TopNav.tsx（新增）
- 实现顶部导航栏。
- 左侧展示 logo 与站点名，右侧按要求顺序展示：模糊搜索、主页、帖子、标签、友链。
- 使用 `a` 标签实现导航语义，保留 SPA 导航体验。

### src/components/Home.tsx
- 移除主页发帖表单，发帖入口迁移至独立编辑页。
- 保留左中右结构（资料、帖子流、时间轴），并新增搜索过滤。
- 帖子正文改为 Markdown 渲染组件展示。
- 保留评论功能。
- 明确项目注释文案：仅 RobinElysia / Meow 管理登录，无注册，访客可浏览评论。

### src/components/PostEditorPage.tsx（新增）
- 新增帖子编辑页：
  - 文档列表（可选中已有帖子）
  - 新建文档
  - 发布（新建）
  - 更新（编辑已有）

  - 插入本地图片到文中任意光标位置（转为 Markdown 图片）
  - 导入本地 `.md` 文件
  - 实时预览（Markdown + LaTeX + Mermaid）

### src/components/MarkdownRenderer.tsx（新增）

### src/components/Login.tsx

## 代码审阅结论

### 1) 业务逻辑是否正确
- 主页发帖已完整抽离到编辑页，首页仅展示与评论。
- 编辑页已具备发布、更新、删除、导入 md、插图与实时渲染能力。
- 帖子详情页可正确展示 Markdown / LaTeX / Mermaid。
- 顶部导航按要求完成并具备页面切换能力。

### 2) 编码乱码风险
- 本次新增与重写文件统一使用 UTF-8 输出，未引入乱码文本。

### 3) 安全性
- 已做：
  - 管理操作接口权限白名单（管理员用户名）
  - 输入归一化与长度限制
  - Markdown 渲染默认不执行原始 HTML
- 仍有残余风险（历史设计）：
  - 登录凭据仍为硬编码，建议下一步迁移至环境变量并改为哈希校验。

### 4) 最佳实践与可读性
- 组件分层更清晰：导航、主页、编辑器、详情、渲染器职责分离。
- Mermaid 采用动态加载，减少非图表场景的首屏负担。
- 接口行为语义明确，代码可读性可维护性提升。

### 5) 项目规整与冗余
- 改动集中在 `App/components/server/types` 主链路，未引入无关目录。
- 移除主页冗余发帖入口，避免重复业务入口导致维护分叉。

## 视觉与规范审查（已结合前端设计与界面规范）
- 保持现有配色与背景氛围。
- 导航与编辑器采用更接近 shadcn 风格的玻璃卡片、边框、层级和交互。
- 增强可访问性：关键输入增加 `aria-label`、登录表单补充自动填充语义。
