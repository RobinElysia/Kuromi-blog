# 1.0.7

发布日期：2026-03-24

## 本次目标
- 修复帖子详情页与编辑实时预览在渲染行为上的不一致。
- 修复详情页点列样式、换行行为与长文本溢出被遮挡问题。

## 修改文件

### `package.json`
### `package-lock.json`
- 新增依赖 `remark-breaks`。
- 目的：使用成熟轮子实现单换行转 `<br>`，让详情页与编辑预览换行行为一致。

### `src/lib/markdown.ts`
- 在统一 ByteMD 插件管线中加入 `remark-breaks`（`breaksPlugin`）。
- 编辑页和详情页都共用 `markdownPlugins`，确保 Markdown 语义渲染一致。

### `src/index.css`
- 列表渲染增强：
  - `ul/ol` 增加 `list-style-position: inside`，避免 marker 被容器边界裁掉。
  - `li::marker` 设置主题色，提升可见性。
- 文本换行增强：
  - 为正文、列表、表格、引用增加 `overflow-wrap: anywhere` + `word-break: break-word`。
  - 链接增加 `word-break: break-all`，防止超长 URL 撑破容器。
- 代码块保持水平滚动（仅 `pre` 横向滚动，不影响普通文本换行）。
- Markdown 面板增加 `box-sizing` 与 `max-width: 100%`，避免内容计算宽度导致超出。

## 兼容性说明
- 未改动后端 API 与业务逻辑（发帖、详情、评论、权限）
- 未改动帖子数据结构；历史内容继续可读。
- 安全策略未放宽，仍使用现有 sanitize 机制。
