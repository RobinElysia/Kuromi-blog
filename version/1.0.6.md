# 1.0.6

发布日期：2026-03-24

## 本次目标
- 修复详情页 Markdown 内容区“颜色突兀”的视觉问题。
- 保持既有配色与背景，统一成更贴合 Kuromi 的半透明层次感。

## 修改文件

### `src/index.css`
- 新增 Kuromi 向 Markdown 颜色变量：
  - `--md-kuromi-surface`
  - `--md-kuromi-border`
  - `--md-kuromi-link`
  - `--md-kuromi-link-hover`
- 为详情 Markdown 容器（`.kuromi-markdown .markdown-body`）增加：
  - 半透明渐变背景（替代生硬深黑块）
  - 柔和粉紫边框
  - 圆角和轻阴影
  - 适当内边距
- 调整详情 Markdown 链接色为低饱和薄荷青，并设置 hover 亮色反馈，避免默认蓝链在 Kuromi 主题里突兀。

## 设计与实现说明
- 未改动业务逻辑，仅做样式层优化。
- 保留编辑器和渲染器功能（Markdown / LaTeX / Mermaid / 图片插入）不变。
- 配色方向遵循现有页面背景，不引入冲突色。
