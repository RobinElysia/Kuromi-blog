# 1.0.13

## 修改目标
- 保持现有站点配色与背景层不变，完成导航栏 `/friends` 对应的友链页面。
- 参考根目录 `Link.md` 补全友链信息，并保证移动端/桌面端响应式展示。
- 保留项目关键说明：无注册入口，仅 RobinElysia 和 Meow 可管理，访客模式可浏览。

- `src/components/Friends.tsx`（新增）

## 具体改动
### 1) 路由接入友链页面（`src/App.tsx`）
- 新增路由类型：`{ type: "friends" }`。
- `resolveRoute("/friends")` 从占位页改为返回 `friends` 路由。
- 在主渲染分支接入 `<Friends search={search} />`，复用顶部搜索输入。

### 2) 新增友链页面（`src/components/Friends.tsx`）
- 基于 `Link.md` 补齐友链数据（名称、头像、描述、链接、标签）。
- 页面整体复用现有组件体系（shadcn 风格基础组件）：`Card`、`Badge`、`Button`。
- 视觉保持与现有站点一致：
  - 保留深色玻璃态卡片语义与玫红/青色渐变高光。
  - 使用 Kuromi 素材（`src/public/kuromi/1x1/illust_2.png`、`logo_gradient_en.svg`）作为页面氛围元素。
- 响应式布局：
  - 头部信息卡在小屏单列、大屏双列。
  - 友链卡片在 `1/2/3` 列自适应。
- 搜索联动：
  - 使用顶部搜索词按 `名称 + 描述 + 标签` 模糊过滤友链。
- 安全与可用处理：
  - 统一链接规范化：无协议域名自动补 `https://`。
  - 拦截未知危险协议（返回 `#` 防止执行）。
  - 外链打开使用 `target="_blank"` + `rel="noreferrer noopener"`。
