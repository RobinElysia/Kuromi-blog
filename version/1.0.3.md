# 1.0.3

## 本次目标
- 保持现有配色与背景，不改主视觉氛围。
- 仅优化Home右侧时间轴控件：
  - 增加动画
  - 调整尺寸
  - 去掉发黑背景容器

## 轮子调研结论
- 已调研可复用方案：`@radix-ui/react-slider` / shadcn `Slider` 等。
- 这些轮子适合直线滑杆，不适合当前“弧线轨迹 + 页面滚动映射 + 亚克力光点”组合需求。
- 因此本轮继续保留现有业务逻辑，采用手写样式与动画增强（不引入额外依赖）。

## 修改文件

### src/components/Home.tsx
- 右侧时间轴容器改为透明样式，移除深色背景卡片（黑底块）。
- 调整控件整体尺寸：
  - 网格列宽从 `180px` 增至 `220px`
  - 弧线区域从 `72x280` 调整为 `132x312`
  - 拖拽点从 `20px` 调整为 `28px`
- 弧线路径改为双层：
  - 底层柔和渐变线
  - 上层流光虚线（用于动画）
- 保留原有滚动映射逻辑，不动业务行为（点击轨道与拖拽光点仍可快速滚动页面）。

### src/index.css
- 新增时间轴动画：
  - `arcFlow`：弧线流光移动
  - `acrylicPulse`：亚克力点呼吸发光
- 新增 `.arc-flow`、`.acrylic-dot` 样式类。
- 增加 `prefers-reduced-motion` 兼容，减少动画时自动禁用动效。

## 代码审阅结论
- 业务逻辑：未删除滚动交互核心逻辑，仅做样式和动效增强。
- 安全性：无新增危险 API，未引入 HTML 注入路径。
- 可读性：动画集中在 `index.css`，交互逻辑仍在 `Home.tsx`，职责清晰。
- 编码：中文文案和源码均为 UTF-8，无乱码。
