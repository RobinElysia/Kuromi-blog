# 1.0.5

发布日期：2026-03-24

## 本次修复目标
- 修复 Markdown 展示区的版式问题：标题样式、段落行距、有序/无序列表符号、整体可读性。
- 增大Home右侧时间轴高度，避免视觉上“太短”。

## 修改文件

### `package.json`
### `package-lock.json`
- 新增依赖：`github-markdown-css`。
- 目的：引入成熟 Markdown 样式轮子，避免手写样式遗漏导致的渲染退化。

### `src/index.css`
- 新增 `@import "github-markdown-css/github-markdown.css"`。
- 在 `.bytemd .markdown-body` 下补充并统一：
  - 正文字号和行高；
  - 标题层级间距与字重；
  - `p/ul/ol/pre/table/blockquote` 段落间距；
  - `ul/ol` 列表样式（恢复 `disc/decimal`）；
  - 嵌套列表间距。
- 保持 Kuromi 现有色盘和背景风格，不改主题方向。

### `src/components/Home.tsx`
- 时间轴高度改为自适应计算：`window.innerHeight * 0.62`，并限制在 `420 ~ 760` 范围。
- 滚动点和刻度位置改为基于动态高度计算，避免高度变化后错位。
- 刻度点增加为 `6` 个（0/20/40/60/80/100），提升长轴可控性。

## 兼容性与业务逻辑说明
- 未删除或改写帖子数据结构、后端 API、权限逻辑。
- 仅修复展示层与交互层样式，不影响历史帖子内容读取。
- 保留“无注册，仅 RobinElysia/Meow 管理，访客可浏览”的业务说明。
